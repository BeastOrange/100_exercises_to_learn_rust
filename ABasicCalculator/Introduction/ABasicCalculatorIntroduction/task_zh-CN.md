# 基础计算器

在本节中，我们将学习如何使用 Rust 作为一个**计算器**。  
这听起来可能不算什么，但这将使我们有机会涵盖 Rust 的许多基础知识，例如：

- 如何定义和调用函数  
- 如何声明和使用变量  
- 基本类型（整数和布尔值）  
- 算术运算符（包括溢出和下溢的行为）  
- 比较运算符  
- 控制流  
- 崩溃处理  

通过几个练习掌握基础知识，将使你更熟练地使用这门语言。  
当我们继续学习更复杂的话题时，比如特性和所有权，你将能够专注于新概念，  
而不会因为语法或其他细节而困惑。

<br/>

## 任务  
入门任务是完成 `intro` 函数。  
该函数应返回字符串：***I'm ready to build a calculator in Rust!***
