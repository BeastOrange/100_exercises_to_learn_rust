## 你好，世界！

现在，让我们编写你的第一个 Rust 程序。在学习新语言时，通常会编写一个小程序，将文本`Hello, world!`打印到屏幕上，所以我们在这里也会这样做！

您可以看到左边的**编辑器**窗口。这是你的沙盒，您可以在这里测试您的代码。`main.rs` 文件有以下几行：

```rust
fn main() {
    // 在这里放置您的代码以运行它
}
``` 
现在让我们将主函数中的行替换为以下代码：

```rust
    println!("Hello, world!");
```
##### 示例：打印 Hello, world! 的程序

要运行这段代码，请点击编辑器左上角的 **Run** 图标。在**任务描述**窗口中点击 **Check** 按钮以运行测试，以检查提供的解决方案的正确性。

如果成功打印了`Hello, world!`，恭喜你！你已经正式编写了一个 Rust 程序。这让你成为了一名 Rust 程序员——欢迎！

### Rust 程序的组成

让我们详细回顾一下你的 Hello, world! 程序中刚刚发生的事情。这里是拼图的第一部分：

```rust

fn main() {

}
```

这些行定义了一个 Rust 中的函数。`main` 函数是特殊的：这是每个可执行 Rust 程序中总是首先运行的代码。第一行声明了一个名为 `main` 的函数，它没有参数且不返回任何值。如果有参数，它们将放在括号内，`()`。

同时，注意函数体被括在大括号 `{}` 中。Rust 要求所有函数体都使用这些括起来。良好的风格是在函数声明同一行放置开括号 `{`，并在它们之间留一个空格。

在撰写本文时，一种称为 `rustfmt` 的自动格式化工具正在开发中。如果你希望在 Rust 项目中保持一种标准风格，`rustfmt` 将以特定风格格式化你的代码。Rust 团队计划最终将此工具与标准 Rust 发行版一起发布，比如 `rustc`。所以视你阅读本书的时间而定，它可能已经安装在你的电脑上！请查看在线文档以获取更多细节。

在 `main` 函数中有以下代码：

```rust
    println!("Hello, world!");
```

这行代码完成了这个小程序中的所有工作：它将文本打印到屏幕上。这里有四个重要的细节值得注意。首先，Rust 的风格是用四个空格缩进，而不是用制表符。

第二，println! 调用了一个 Rust 宏。如果它调用的是函数，则输入为 `println`（没有 `!`）。我们将在后面更详细地讨论 Rust 宏。目前，你只需知道使用 `!` 表示你正在调用一个宏，而不是一个普通函数。

第三，你看到 `"Hello, world!"` 字符串。我们将此字符串作为参数传递给 `println!`，字符串会被打印到屏幕上。

第四，我们以分号（`;`）结尾，表示这个表达式已经结束，下一个可以开始。大多数 Rust 代码行以分号结束。
