欢迎来到 JetBrains 学院对令人兴奋的 [rustlings](https://github.com/rust-lang/rustlings) 课程的改编版本。

本课程包含小型练习，以便让您习惯于阅读和编写 Rust 代码。这包括阅读和响应编译器消息！

原始材料由 [Carol Nichols](https://github.com/carols10cents) 撰写。这个改编版本尽量紧扣 rustlings 的最初精神，但在 IDE 内部添加了一些关于 Rust 开发的有用提示。

对于首次学习 Rust 的人，还有其他一些资源可供选择：
* [The Book](https://doc.rust-lang.org/book/index.html) – 由 Steve Klabnik 和 Carol Nichols 编写，是学习 Rust 的最全面的资源，并有 Rust 社区的贡献。在整个课程中，会提供相关章节的链接，以更基础的方式描述所讨论的概念。

* [Rust By Example](https://doc.rust-lang.org/rust-by-example/index.html) – 一个在线可运行示例的集合，展示了各种 Rust 概念和标准库。
