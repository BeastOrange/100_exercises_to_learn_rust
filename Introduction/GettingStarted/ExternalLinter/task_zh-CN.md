## 启用外部代码检查器

IntelliJ Rust 插件不能检测所有错误。它依赖于 Rust 编译器来完成这些工作。在您学习 Rust 的过程中，能够在输入时看到错误是很有帮助的。为了实现这种效果，我们建议启用外部代码检查器，如下所示：

1. 前往 **设置 / 偏好设置 | 语言与框架 | Rust | 外部代码检查器**（如果在 RustRover 中打开，可以直接选择 **设置 / 偏好设置 | Rust | 外部代码检查器**）。
2. 设置参数如下：
    - 在 **外部工具：** 列表中选择 **Cargo Check**；
    - 勾选 **运行外部代码检查器以实时分析代码** 复选框。

![外部代码检查器](images/external-linters.png)
3. 按下 **确定**。

完成这些操作后，%IDE_NAME% 将报告 IntelliJ Rust 插件或 Rust 编译器检测出的所有错误。
