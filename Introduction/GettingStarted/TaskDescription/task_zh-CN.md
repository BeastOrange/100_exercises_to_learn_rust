## 任务描述

**任务描述** 窗口为您提供完成任务所需的所有信息：

对于理论任务，描述提供学习和阅读材料。
对于测验，它提供多项选择题。
对于编程作业，它陈述需要解决的问题。

使用任务描述窗口元素执行以下操作：

| 元素                                                            | 描述                                                                                                       |
|----------------------------------------------------------------|-----------------------------------------------------------------------------------------------------------|
| **检查**                                                       | <font size="-1">检查您的答案是否正确（对于测验）或您的代码解决方案（对于编程任务）</font>                    |   
| **运行**                                                       | <font size="-1">运行您的代码（用于理论任务）</font>                                                        |
| ![](images/back.svg)                                           | <font size="-1">返回到上一个任务</font>                                                                  |
| ![](images/forward.svg) &nbsp;<font size="-1">或</font> **下一步** | <font size="-1">前往下一个任务</font>                                                                     |
| ![](images/reset.svg)                                          | <font size="-1">放弃您在任务中所做的所有更改，重新开始</font>                                             |
| ![](images/commentTask.svg)                                    | <font size="-1">对特定任务留下评论</font>                                                                 |
| <a>查看解决方案...</a>                                        | <font size="-1">揭示正确答案并显示 <b>差异</b></font>                                                     |

我们建议保持任务描述窗口可见，而不要完全隐藏它。如果过于分散注意力，可以通过点击任务描述窗口右上角的 ![](images/hideToolWindow.svg) 按钮来隐藏它。

如果您使用两个显示器，可能需要将任务描述面板切换到悬浮模式并移动到第二个显示器，或者将其放置在主 IDE 窗口附近。要执行此操作，请点击工具窗口设置 ![](images/gear.svg) 图标：
