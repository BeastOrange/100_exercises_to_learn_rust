## 尾声

我们的 Rust 之旅到此结束。  
我们已经覆盖了很多内容，但仍然远未详尽：Rust 是一门有着广泛“表面”和更大生态系统的语言！  
但不要被吓到：**没有必要学会所有东西**。  
当你在处理项目（如后端开发、嵌入式系统、命令行界面（CLI）、图形界面（GUI）等）时，你会自然地学会所需的一切。

最终，没有捷径：如果你想精通某件事，你需要一遍又一遍地去实践。在本课程中，你写了相当多的 Rust 代码，足够让语言及其语法在你手指间流畅运转。但要完全掌握它，还需要写更多的代码。然而，如果你坚持练习，这一刻必然会到来。

## 更进一步

让我们以一些额外资源的指南结束，为你在 Rust 学习之旅中提供更多帮助。

### 练习

你可以在 [`rustlings`](https://github.com/rust-lang/rustlings) 项目和 [exercism.io](https://exercism.io) 的 Rust 路线上找到更多练习来练习 Rust。

### 入门材料

如果你希望以不同的视角来理解我们在课程中涉及的概念，可以查看 [The Rust Book](https://doc.rust-lang.org/book/title-page.html) 和 ["Programming Rust"](https://www.oreilly.com/library/view/programming-rust-2nd/9781492052586/)。  
你一定会学到新东西，因为它们并不完全涵盖相同的主题；Rust 的“表面”确实广阔！

### 高级材料

如果你想深入研究这门语言，可以参考 [Rustonomicon](https://doc.rust-lang.org/nomicon/) 和 ["Rust for Rustaceans"](https://nostarch.com/rust-rustaceans)。  
["Decrusted" 系列](https://www.youtube.com/playlist?list=PLqbS7AVVErFirH9armw8yXlE6dacF-A6z) 也是学习许多最受欢迎的 Rust 库内部机制的优秀资源。

### 领域特定的材料

如果你想使用 Rust 进行后端开发，  
可以查看 ["Zero to Production in Rust"](https://zero2prod.com)。  
如果你想使用 Rust 进行嵌入式开发，  
可以查看 [Embedded Rust Book](https://docs.rust-embedded.org/book/)。

### 精修课程

你还可以找到跨领域的关键主题资源。  
关于测试，可以参考  
["Advanced testing, going beyond the basics"](https://rust-exercises.com/advanced-testing/)。  
关于遥测，可以参考 ["You can't fix what you can't see"](https://rust-exercises.com/telemetry/)。
