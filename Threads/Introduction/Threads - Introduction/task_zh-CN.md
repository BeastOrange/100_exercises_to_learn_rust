Rust 的一个重要承诺就是_无畏并发_：让编写安全并发程序变得更加容易。  
到目前为止，我们还没有真正接触到这一点。我们所做的所有工作都是单线程的。  
现在是改变的时候了！

在本章中，我们将使票务商店支持多线程。  
我们将有机会接触到 Rust 的大多数核心并发功能，包括：

- 使用 `std::thread` 模块的线程  
- 使用通道的消息传递  
- 使用 `Arc`、`Mutex` 和 `RwLock` 的共享状态  
- `Send` 和 `Sync`，这两个特性编码了 Rust 的并发保证  

我们还将讨论多线程系统的各种设计模式及其一些权衡。
